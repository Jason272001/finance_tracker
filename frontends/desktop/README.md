# KeeperBMA Desktop Frontend

Desktop client installers.

## Windows

Installer executable:
- `KeeperBMA-Web-Installer.exe`

What it does:
- Creates desktop shortcut `KeeperBMA Web.lnk`
- Opens app in dedicated app-style window (`--app=...`) using Chrome/Edge

Default URL:
- `https://jason272001.github.io/finance_tracker/frontends/web/`

## macOS

Click installer package:
- `KeeperBMA-Web-macOS.app.zip`

Install (click only):
1. Download `KeeperBMA-Web-macOS.app.zip`
2. Double-click zip to extract
3. Drag `KeeperBMA Web.app` to `Applications`
4. Double-click `KeeperBMA Web.app` to run

What it does:
- Opens KeeperBMA in a dedicated app-style browser window
